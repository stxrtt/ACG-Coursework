Hello! This readme assumes you have NodeJS installed in your system.
Open this directory in a terminal and run the following command:

node server.js

Then open a browser and go to the following URL:

http://localhost:8080/

The simulation uses default mouse orbit controls from the OrbitControls package:
- Left mouse button: rotate
- Right mouse button: pan
- Scroll wheel: zoom

Click the green control points to move them around and deform the dodecahedron.
